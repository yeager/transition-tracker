# Transition Tracker

Track library transitions in Debian and Ubuntu.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install transition-tracker
```
